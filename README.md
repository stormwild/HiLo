# HiLo

Simple java guessing game using random numbers.

## Pre-requisites

- Java
- Visual Studio Code
- Git

## Setup

- Install vscode
- Install the [recommended extensions](https://code.visualstudio.com/docs/languages/java)
- Install Maven for Java
- Clone the repository 
- Open in vscode
- Run in integrated terminal
